# Basic
Аннотации типов: Как указать ожидаемый тип для аргумента и для возвращаемого значения функции? Занимается ли интерпретатор Python проверкой этих типов во время выполнения (runtime)?
Лямбда (синтаксис): Что такое лямбда-функция и как её определить? Может ли у неё не быть аргументов (если да — какой синтаксис)?
Ограничения лямбд: Сколько выражений (expressions) может содержать тело лямбда-функции? Может ли она содержать инструкции (statements), например print или raise?
Практика лямбд: Напиши две лямбды: одна принимает a и b и возвращает их сумму, вторая не принимает ничего и всегда возвращает строку "hello".
Рекурсия: Ограничено ли в Python количество вложенных рекурсивных вызовов? Что произойдет при превышении лимита?

# Advanced
Опасные дефолтные значения: Почему код def func(x, items=[]): items.append(x) считается классической ловушкой? Что именно происходит с объектом items при повторных вызовах функции? Как переписать этот код правильно?
Эмуляция static переменных: Как в Python реализовать поведение, аналогичное ключевому слову static в C++ (чтобы переменная внутри функции сохраняла состояние между вызовами), не используя глобальные переменные? (Подсказка: атрибуты функции).
Интроспекция через inspect: Какие возможности дает модуль inspect для анализа сигнатуры функции (параметры, аннотации, значения по умолчанию)? В каких случаях это может быть полезно при разработке фреймворков?
Захват состояния: (Связующий вопрос к теме Scopes) Если лямбда создается внутри цикла, какое значение внешней переменной она «увидит» при вызове — то, которое было при создании, или то, которое стало в конце цикла?
Асинхронные функции (Async): Как определить асинхронную функцию? В чем её главное отличие от обычной с точки зрения вызова (что возвращается при прямом вызове func())?
Смешивание async и sync: Может ли асинхронная функция вызывать обычные функции? А наоборот — может ли обычная функция вызвать асинхронную без специальных инструментов (event loop)?
