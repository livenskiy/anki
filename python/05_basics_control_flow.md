# Basic
Индексы в циклах: Почему в Python считается плохой практикой использовать range(len(sequence)) для итерации и какой инструмент вместо этого является стандартным?
Цикл for...else: В каком случае выполняется блок else после цикла? Выполнится ли он, если тело цикла ни разу не отработало (пустая итерация)? Работает ли эта логика для цикла while?
Синтаксис Comprehensions: Является ли выражение [for x in numbers] валидным? Чего в нем не хватает?
Фильтрация в списке: Куда именно в выражении [x for x in numbers] нужно вставить условие, чтобы оставить только положительные числа?
Тернарный оператор в цикле: Как с помощью спискового включения заменить все отрицательные числа в списке на 0, оставив положительные как есть? Обязателен ли здесь блок else?

# Advanced
Вложенные циклы (Flattening): У нас есть список списков [[1, 2], [3, 4]]. Напиши списковое включение, которое «выпрямит» его в один плоский список. В каком порядке должны идти операторы for?
Обработка неоднородных данных: Код [x for g in [[1, 2], 3, [4]] for x in g] падает с ошибкой 'int' object not iterable. Как модифицировать это выражение (или добавить проверку), чтобы оно корректно обработало и списки, и отдельные числа?
Множественные условия: Сколько операторов if можно поставить в конце спискового включения? Как они будут интерпретироваться (как or или как and)?
Сложные конструкции: Если в одном выражении используется несколько вложенных for, можно ли поставить фильтрующий if после каждого из них? Как это повлияет на логику?
