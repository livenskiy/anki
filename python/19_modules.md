# Basic
Модули vs Пакеты: В чем техническое различие? Обязателен ли __init__.py в современных версиях Python, чтобы папка считалась пакетом?
Механика импорта: Как импортировать конкретные объекты, как переименовать их при импорте (as) и в каком namespace они оказываются?
Импорт «звездочкой» (from module import *): Как это сделать и почему в реальных проектах это считается плохой практикой?
Поиск модулей: В каком порядке Python ищет модуль при import name? Где хранится список путей и как добавить туда свой путь программно?
Кэширование: Сколько раз выполнится код модуля, если импортировать его в разных частях программы несколько раз? Можно ли легко обновить модуль без перезапуска программы?
Namespace-пакеты: Если мы просто хотим раскидать файлы по папкам, обязательно ли делать из них пакеты?
Синтаксис import vs from: В чем разница в результате между import pkg.mod.obj и from pkg.mod import obj?

Главное ограничение относительных импортов: Почему они часто ломаются при запуске скрипта напрямую (python script.py) и как это лечит запуск через -m?
