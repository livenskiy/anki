# Basic
Произвольные аргументы: Как определить функцию, которая принимает любое количество позиционных аргументов? А любое количество именованных (keyword) аргументов?
Keyword-only аргументы: Как заставить пользователя передавать определенные аргументы только по имени? В каких ситуациях это полезно для дизайна API?
Асинхронные функции (Async): Как определить асинхронную функцию? В чем её главное отличие от обычной с точки зрения вызова (что возвращается при прямом вызове func())?
Смешивание async и sync: Может ли асинхронная функция вызывать обычные функции? А наоборот — может ли обычная функция вызвать асинхронную без специальных инструментов (event loop)?

# Advanced

Дизайн через Positional-only: Зачем ввели этот синтаксис? Приведи хотя бы две причины (подсказки: обратная совместимость при переименовании аргументов и исключение конфликтов с **kwargs).

Различие между / и *: В чем концептуальная разница между «всё, что до /» и «всё, что после *» в определении параметров функции?
