Прогресс повторения хранится в tracker.txt

Номер означает степень освоения материала:
.1 — Повтор через 1 день
.2 — Повтор через 3 дня
.3 — Повтор через 7 дней
.4 — Повтор через 14-21 день
.5 — Повтор через 1-2 месяца
.X - Тема завершена

Полезные приёмы:
- Просить после прохождения тестов по разделу рассказать о своих слабых сторонах и дать доп.вопросы
- Просить показать три утверждения каждое из которых может быть ложным и дать ответы
- Задачи на рефакторинг "Дай мне задачу, где я должен оптимизировать память миллиона объектов с помощью __slots__ и замерить разницу"
- Просить дать код с ошибкой, например с перепутанным static и class методами или неправильным дескриптором который хранит данные сам
